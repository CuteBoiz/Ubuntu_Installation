## POPULAR PACKAGE FOR SUBLIME TEXT

### 1. Package Control

*This is the most important package*

**Install:**

```sh
[Ctrl + Shift + P] => [Install Package Control]

```

**Install A Package**

```sh
[Ctrl + Shift + P] => [Package Control: Install Package] => "Name of Package"

```
